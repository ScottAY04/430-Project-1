Countries API Doc

/allCountries

Query Params

None

Returns

Json array of a country object

Format: [{"name": string, "capital": string, "finance": { "currency": string, "currency_name": string, "currency_symbol": string }, "region": string, "subregion": string, "nationality": string, "timezones": [ { "zoneName": string, "gmtOffset": number, "gmtOffsetName": string, "abbreviation": string, "tzName": string } ], "latitude": string, "longitude": string}]

/getCountry

Query Params

name: the country name

Returns

Json array of a country object

Format: [{"name": string, "capital": string, "finance": { "currency": string, "currency_name": string, "currency_symbol": string }, "region": string, "subregion": string, "nationality": string, "timezones": [ { "zoneName": string, "gmtOffset": number, "gmtOffsetName": string, "abbreviation": string, "tzName": string } ], "latitude": string, "longitude": string}]

/getCountryByContinent

region: the continent/region name

Returns

{"name": string}

/getCountryByLetter

Query Params

Returns

{"name": string}
